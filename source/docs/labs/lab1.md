# Лабораторная работа 1: Создание статического сайта

## Цель работы
Освоить процесс создания статического сайта с использованием генератора MkDocs и научиться публиковать его на GitHub Pages.

## Задание
1. Создать репозиторий на GitHub
2. Установить и настроить MkDocs
3. Создать структуру сайта
4. Настроить тему оформления
5. Опубликовать сайт на GitHub Pages

## Ход работы
1. Создан репозиторий `yanavlitv.github.io`
2. Установлен MkDocs
3. Создана структура сайта:
   - Главная страница
   - Страница "Об авторе"
   - 1 страница лабораторной работы
4. Настроена навигация в файле `mkdocs.yml`
